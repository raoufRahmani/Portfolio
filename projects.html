Paris House Price Prediction

Objective

Study the effect of features like number of rooms and surface area on house prices in Paris using multiple linear regression and improve predictions with ExtraTreesRegressor.

Dataset Description

Uses the DVF dataset from French public finance, containing detailed real estate transaction data across Paris.

Exploratory Data Analysis

Checking missing values, data types, visualizing distributions, and analyzing the number of main rooms.

Modeling

Estimated econometric models, interpreted regression results, then enhanced price prediction with ExtraTreesRegressor assessing variable importance.

Used Cars Market in France — Analysis and Price Prediction

Overview

Analyzed used cars market in France via deep exploratory data analysis on a dataset scraped from major websites, followed by price prediction using XGBoost Regressor.

Data Cleaning and Preparation

Removed unwanted characters (/, €), converted columns to correct types, handled missing values, and ensured dataset consistency.

Exploratory Data Analysis

Investigated popular brands and models, impact of mileage and age on price, and variations by fuel type and gearbox.

Machine Learning Model

Built a regression model with XGBoost and used OneHotEncoding for categorical features. The model offers reasonable price predictions and insights on feature importance.

Amazon Scraper App

Purpose

This Streamlit web application allows users to scrape Amazon product data including names and prices, and export the result to a CSV file.

Features

Users enter the Amazon URL, number of pages to scrape, and a desired filename. The app extracts structured product data and saves it locally.

Technologies

Built using Python (requests, BeautifulSoup, pandas), deployed via Streamlit Cloud.
